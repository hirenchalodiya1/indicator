Requirnments
---------------
python3, pipenv
---------------

Run following command to setup project
----------------------------------
pipenv install
pipenv shell
----------------------------------

For running program
-------------------------------
python3 broker.py
-------------------------------


Notes:
1.
    If during runtime if it generates ```FileError```, then ignore it and run command till error doesn't appear.
    This error implies that data coudn't fetch from internet as cerebro engine reads data from live feeds so re run command.
2.
    Ignore RunTime Warning of matplotlib as graphics are can vary system to system.
